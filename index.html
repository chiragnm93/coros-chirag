Personal Training Assistant

A personal AI assistant and associated system for serious age-group endurance athletes.

What it does

After each training session, COROS activity data (heart rate zones, pace, power, GPS) is automatically ingested into a locally-hosted AI training system. The system tracks physiological adaptations, manages fueling and sodium protocols, and generates personalised training recommendations aligned to Chirag's specific race targets and physiology requirements.

Current race targets

Data handling

All data is stored locally and privately. No data is shared with third parties, sold, or used commercially. This system has one user: the applicant. ME.

COROS Integration

This project uses the COROS API to sync activity data post-session via the Workout Summary Data Push Service. COROS devices are the primary data source for this coaching system.
